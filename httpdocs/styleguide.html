The Chaincode site uses Foundation as its baseline. This provides us with a simple grid system, and some basic default styling. You can find out a lot more about Foundation by taking a look at their docs.

This is an example of our paragraph styling. It uses Open Sans via the free Google Web Fonts library. This font is 300 weight, #474849 in color and displayed at 1.125em in size.

					
	font-family: 'Open Sans', serif;
	font-weight: 300;
	color: #474849;
	font-size: 1.125em;
	line-height: 1.8;

Our intro section uses a header also in Open Sans. This font is 600 in weight, #929497 in color and displayed at 2.25em in size.

					
	font-family: 'Open Sans', sans-serif;
	font-weight: 600;
	font-size: 2.25em;
	margin: 0;
	color: #929497;

The background image in our intro is a pattern based off the logo designs. You can see the full image here.

Header in Montserrat

We also have a header using the Montserrat font also from the Google Web Fonts library. This font is 700 weight, #FFFFFF in color, displayed at 2.75em with a background color of #FFDA0D.

					
	font-family: 'Montserrat', sans-serif;
	text-transform: uppercase;
	font-weight: 700;
	color: #fff;
	text-align: center;
	margin: 0 0 1em;
	padding: 0.75em 0;
	background: #ffda0d;
	font-size: 2.75em;
